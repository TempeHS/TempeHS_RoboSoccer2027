# 04 - Kiwi Drive Tuning Worksheet (Year 9)

Use this worksheet while tuning your 3-wheel Kiwi drive.

How to use:

1. Pick one test motion (for example forward only).
2. Start with small gains.
3. Change one value at a time.
4. Write what happened in observations.

Tip:

- vx = left-right movement
- vy = forward-back movement
- w = turn/rotation

- Ball tracking still works while moving
